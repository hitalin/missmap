# missmatch

**Fediverse ユーザーのためのサーバーマッチングサイト**

日本の Misskey サーバーの連合関係を地図で可視化し、自分に合うサーバー選びをサポートします。

## コンセプト

日本には数百の Misskey サーバーが存在しますが、どのサーバーを選ぶかで体験が大きく変わります。

- どのサーバーがどの界隈に属している？
- そのサーバーに入ると、どこと繋がれて、どこと断絶する？

missmatch は、これらの情報を **地図** として可視化します。判断するのはあなた自身です。

## 機能

### 界隈マップ

サーバー同士の繋がりをグラフで可視化。

- サーバー間の連合関係を一目で把握
- コミュニティのクラスタリング（界隈）を自動検出
- 「このサーバーはどの界隈？」がわかる

### ブロックマップ

サーバー間の断絶・ブロック関係を可視化。

- 「このサーバーに入ると誰と会えなくなる？」がわかる
- 界隈マップの裏側として、断絶の全体像を把握

## 対象

**日本の Misskey サーバー**に特化しています。

- Misskey / Misskey Fork
- 主に日本語でコミュニケーションが行われているサーバー
- 公開されている情報のみを使用

## やらないこと

- **審判にならない**: 「良いサーバー」「悪いサーバー」を決めつけない
- **監視しない**: ユーザーの投稿内容や行動を追跡しない
- **プライベート情報を扱わない**: 公開 API で取得できる情報のみ使用

## 技術スタック

- **Framework**: SvelteKit 5
- **Graph**: Cytoscape.js
- **Database**: Cloudflare D1
- **Deploy**: Cloudflare Pages
- **Language**: TypeScript
